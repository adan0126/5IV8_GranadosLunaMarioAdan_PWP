Memoization - Exercise Hard

Descripción del Ejercicio

Este ejercicio implementa una función memoize que utiliza caché para optimizar el rendimiento de funciones costosas. La memoización almacena los resultados de llamadas anteriores y los reutiliza cuando se llama la función con los mismos argumentos, evitando cálculos redundantes.

¿Qué se implementó?

  • Función memoize(): Retorna una versión envuelta de la función original que mantiene caché
  • Estructura de caché con Map: Utiliza Maps anidados para guardar resultados basados en argumentos
  • Optimización de rendimiento: Reduce llamadas innecesarias a funciones costosas
  • Soporte para múltiples argumentos: Funciona con funciones que reciben varios parámetros
  • Manejo de objetos: Los objetos se consideran diferentes a menos que sean la misma instancia en memoria

Conceptos Clave

Caché

Almacenamiento temporal de resultados para acceso rápido

Memoización

Técnica de optimización que reutiliza resultados previos

Higher Order Functions

Funciones que reciben o retornan otras funciones

Closure

La función memoizada tiene acceso al cache gracias a closures
